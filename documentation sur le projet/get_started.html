IMAGE LOG - Get Started

  • Générer une carte
  • Naviguer dans une carte

L'objectif de ce document est de proposer un guide de démarrage pour la génération et de visualisation d'une carte

Génération

1. Extraction de l'outil

Extraire l'archive IMAGELOG.zip

Elle contient les éléments suivants :

imagelog.jar
input/
    imdb_mini.txt
viz/
    index.html
    jquery-1.11.0.min.js
    style.css
    viz.js
    mosaic/

2. Lancement de la génération

Ouvrir un Terminal. Se placer dans le dossier extrait et exécuter la ligne de commande suivante :

$java -jar imagelog.jar -l ./input/imdb_mini.txt -o ./viz/mosaic/

Cette commande lancera la génération à partir du fichier de log imdb_mini.txt et enregistrera les mosaïques dans le dossier ./viz/mosaic/

3. Visualisation

Une fois la génération terminée, ouvrir le fichier ./viz/index.html avec un navigateur web.

Une fenêtre s'ouvre. Zoomez sur une zone en cliquant dessus. Dezoomez en cliquant sur le bouton ZoomOut et revenez au zoom initial en cliquant sur Reset.
